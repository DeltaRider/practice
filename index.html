github cloning steps

  1. go to github.com, log in
  2. make a new repository
  3. copy the https .git clone link
  4. in your terminal navigate to where you want that github repository downloaded to
  5. git clone the https .git link
  6. cd into that github repository that you just cloned
  7. now you can make files
  8. when you're done, you do git add .
  9. git commit -m 'first commit'
  10. git push (or git push origin master
